# **Additional Experiments for ICML 2026 Submission**

This repository provides supplementary experimental results to support the rebuttal of our ICML 2026 submission.

---

## **Contents**

- `HIA_add_experiments.pdf`  
  Contains additional experiments requested by the reviewers, including extended evaluations and ablation studies.

---

## **Purpose**

The goal of this repository is to ensure transparency and reproducibility during the rebuttal phase. All results included here are **additional** to those reported in the main paper and are intended to address reviewer concerns.

---

## **Notes on Anonymity**

To comply with the ICML double-blind review policy:

- The repository is anonymized and does not contain any identifying information about the authors or their affiliations.  
- Any references to prior work are presented in a neutral manner.  
- File names and descriptions are kept generic to avoid revealing identity.  

---

## **Usage**

No code execution is required. Please refer directly to the PDF file for detailed experimental results and analysis.

---

## **Disclaimer**

This repository is intended solely for the purpose of the ICML 2026 review process. It may be updated during the rebuttal period to include further clarifications if necessary.
